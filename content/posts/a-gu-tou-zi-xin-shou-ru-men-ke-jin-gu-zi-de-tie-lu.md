# A股投资新手入门：刻进骨子的铁律

---

## 一、核心理念：复利才是王道

### 1.1 什么是真正的复利

```
亏损10% → 需要11.1%回本
亏损20% → 需要25%回本
亏损50% → 需要100%回本
亏损80% → 需要400%回本
```

**亏钱比赚钱容易。保住本金，就是最大的进攻。**

### 1.2 复利的奇迹

```
2万起步，每月10%：

第1年：2万 → 6万
第2年：6万 → 18万
第3年：18万 → 54万
第4年：54万 → 162万
第5年：162万 → 486万
第6年：486万 → 1458万
第7年：1458万 → 4374万
第8年：4374万 → 1.31亿

8年6500倍
```

**目标曲线**：斜着向上 OR 平着再斜着向上

## 二、五大铁律（必须刻进DNA）

### 铁律一：拒绝上头交易

```
上头信号：
□ 连续亏损后想快速回本
□ "这次不一样"
□ 盈利后想加仓重仓
□ 亏损后不舍得止损

应对：
1. 连续亏损后停止交易
2. 可买可不买时坚决不买
3. 盘后制定计划，盘中只执行
4. 不要想着翻本，该割就割
5. 拒绝盘中临时起意
6. 大赚后减仓
```

### 铁律二：绝不满仓

```
单只股票 ≤ 30%
总仓位 ≤ 80%
永远保留20%以上现金

持仓数量：
新手2-4只，体量大4-8只

❌ 满仓梭哈 = 赌博 = 必输
```

### 铁律三：只做主升浪，只做龙头

```
主升浪：
✓ 均线多头排列
✓ 成交量放大
✓ 股价新高
✓ 回调不破均线

龙头：
✓ 板块涨幅第一
✓ 涨停最早，封板最稳
✓ 有题材、有流动性
✓ 回调抗跌，反弹领先

❌ 抄底 = 亏钱
❌ 杂毛 = 亏钱
```

### 铁律四：小盈小亏，大盈小亏

```
常态：
✓ 小盈利5%-15%
✓ 小亏损-3%以内
✓ 大盈利20%+（偶尔）

避免：
❌ 大亏损-7%以上

卖出：
- 涨幅15%，减仓一半
- 跌幅-7%，立刻止损
- 跌破10日线，清仓
```

### 铁律五：大亏后立刻空仓

```
触发条件：
□ 单月亏损10%
□ 连续5日亏损
□ 大盘连续下跌10%

做法：
1. 立刻停止交易
2. 不要想翻本
3. 空仓休息1-2天
4. 小仓位重新开始

❌ 急于翻本 = 亏更多
❌ 加大仓位 = 找死
```

## 三、新手成长路径

### 3.1 第一阶段：用小钱练纪律（1-2年）

```
资金：2-5万

目标：
- 不管什么模式先选一种
- 形成严格纪律
- 练出稳定心态

标准：
- 不亏大钱 → 合格
- 连续6个月不亏 → 进阶
```

### 3.2 第二阶段：找到自己的交易模式（1年）

```
资金：2-5万

目标：
- 确认模式稳定盈利
- 优化买卖点
- 建立交易系统

标准：连续12个月稳定盈利，回撤10%以内

三种模式选一种精通：

【打板】专注首板/2板/3板，次日不涨停必走
【龙头】专注总龙头/补涨龙，龙头不倒持股不动
【趋势】专注趋势票，跌破趋势线走人

✓ 找到一种模式，不断重复
✓ 模式内一致，模式外不动
```

### 3.3 第三阶段：复利增长

```
资金：10万

前提：模式验证稳定盈利

策略：
- 赚钱加大仓位
- 亏钱减小仓位
- 追求稳定复利

心态：每月10%即可，坚持复利，只是时间问题
```

